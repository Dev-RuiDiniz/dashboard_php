# ROTEIRO DE HOMOLOGAÇÃO

## Checklist por módulo

- [ ] Autenticação (login/logout/reset/lockout)
- [ ] Famílias e ficha social
- [ ] Crianças
- [ ] Entregas por evento
- [ ] Equipamentos
- [ ] Dashboard
- [ ] Relatórios (CSV/XLSX/PDF)
- [ ] Configuração e usuários
- [ ] LGPD (consentimento + auditoria)
- [ ] Fechamento mensal
- [ ] Relatório oficial (SHA256)
- [ ] Histórico e gráficos

## Critério de aprovação

A homologação é aprovada com fluxo ponta a ponta funcional, RBAC coerente, exportações PDF válidas e testes automatizados verdes.
